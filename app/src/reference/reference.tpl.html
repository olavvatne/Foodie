Frameworks

AngularJS

Open source JavaScript web application framework for developing single-page applications. Applications are developed using the MVC pattern, decoupling DOM manipulations from application logic. AngularJS also provide a way to extend the html syntax using directives, which enables developers to create reusable web components.

Compass

Open source CSS authoring framework

Bower

Package management system for client-side web development.

Grunt

Task runner that automate tasks like minification and compilation.
